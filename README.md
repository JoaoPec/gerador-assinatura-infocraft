# Gerador de Assinatura Infocraft

HTML simples para gerar uma imagem PNG de assinatura de e-mail da Infocraft.

## Como usar

1. Abra `gerador-assinatura.html` no navegador.
2. Preencha nome, cargo, e-mail e telefone.
3. Envie a foto e o QR code.
4. Clique em **Baixar imagem**.
5. Insira o PNG baixado no editor de assinatura do e-mail.

O projeto usa apenas HTML, CSS e JavaScript puro.
