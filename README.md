# Gerador de Assinatura Infocraft

HTML simples para gerar assinatura de e-mail da Infocraft.

## Como usar

1. Abra `index.html` no navegador.
2. Preencha nome, cargo, e-mail e telefone.
3. Envie a foto e o QR code.
4. Clique em **Copiar HTML/CSS** e cole no editor de assinatura do e-mail (recomendado — mantém links clicáveis).

### Se o cliente de e-mail não aceitar HTML

Tire um **print** só da área branca da prévia (a assinatura completa):

- **Windows:** `Win + Shift + S` (área) ou Ferramenta de Captura
- **Mac:** `Cmd + Shift + 4` (área)

Cole a imagem no editor de assinatura.

O projeto usa apenas HTML, CSS e JavaScript puro.
